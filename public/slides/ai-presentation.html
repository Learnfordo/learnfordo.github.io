基础认知

什么是大模型

大模型全称「大语言模型」（Large Language Model，简称 LLM），基于 Transformer 框架沿 GPT 路线训练，通过海量文本数据学习语言规律。

💬 Chat 对话

日常问答、写作、翻译、知识问答

💻 AI Coding

代码生成、Debug、代码解释、重构

🦞 AI Agent

大模型是如何进行回复的呢

大模型只会做一件事——就是做文字接龙（预测下一个 Token）

📥 输入文本
"今天天气真"
→
✂️ 分词 Tokenize
[今, 天, 天气, 真]
→
🧠 模型推理
Transformer 计算
→
🔢 计算分数
每个候选 token 打分
→
📊 概率分布
Softmax 输出
每个候选词的概率预测：
按概率"摇骰子"随机选 → 概率越高的词越容易被选中 → 选到"好"后，拼入原文继续预测下一个…

🔑 核心原理

给定上下文，模型通过 Self-Attention 理解每个词之间的关系，最终输出一个概率分布，选择最可能的下一个 token

📖 Token 词表

Token（词元）是能够表达语言中完整语义的最小单元。每个模型都有自己的词表，就像人类的"词典"一样。

从 Chatbot 到 Agent

大模型能力很强，但仍有边界

📏 上下文窗口有限
无法一次性塞入所有信息，长对话会"遗忘"前面的内容
🌍 知识有边界
训练数据是静态的，无法掌握所有最新信息
如何突破限制？→ Context Engineering（上下文工程）

🗣️ Prompt Engineering

用精准的指令引导模型输出

🔍 RAG

检索增强生成，从外部知识库注入上下文

🔧 Function Calling

让模型调用外部工具/API，获取实时数据并执行操作
