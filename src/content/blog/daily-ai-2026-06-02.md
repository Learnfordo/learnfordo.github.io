## 📚 arXiv 论文精选

### 1. 缓解多模态 LLM-as-a-Judge 的感知判断偏差
**Mitigating Perceptual Judgment Bias in Multimodal LLM-as-a-Judge via Perceptual Perturbation and Reward Modeling**
📄 [arXiv:2606.02578](https://arxiv.org/abs/2606.02578) · ICML 2026

**问题：** 多模态 LLM 作为自动评估器时，当视觉证据与文本线索冲突，倾向于奖励"听起来合理"而非"视觉正确"的回答——即感知判断偏差。
**方法：** 提出感知扰动判断数据集（PPJD），通过构建最小编辑的反事实回答来隔离感知错误；设计基于 GRPO 的结构化奖励 + 批量排序训练框架。
**效果：** 在多个 MLLM-as-a-Judge 基准上显著提升感知保真度和排序一致性。

### 2. AdaCodec：视频 MLLM 的预测性视觉编码
**AdaCodec: A Predictive Visual Code for Video MLLMs**
📄 [arXiv:2606.02569](https://arxiv.org/abs/2606.02569)

**问题：** 现有视频 MLLM 将每帧独立编码为 RGB 图像，导致大量时序冗余。
**方法：** 提出预测性视觉编码——仅在场景无法从上下文预测时发送完整参考帧，否则编码帧间变化为紧凑 P-token。
**效果：** 在 Qwen3-VL-8B 上，仅用 1/7 的 token 预算（32k vs 224k），在所有长视频基准上超越基线；首 token 延迟从 9.26s 降至 1.62s。

### 3. ClinEnv：面向 LLM Agent 的交互式多阶段临床环境
**ClinEnv: An Interactive Multi-Stage Long Horizon EHR Environment for Agents**
📄 [arXiv:2606.02568](https://arxiv.org/abs/2606.02568)

**问题：** 现有静态医学基准无法评估 LLM 在真实临床场景中的信息获取和序列决策能力。
**方法：** 构建 ClinEnv，基于真实住院病例，让 LLM 扮演主治医师在多阶段中主动查询 4 个专业 Agent 并做出不可逆决策。
**发现：** 最强模型决策 F1 仅 0.31；出院诊断 F1 (0.51) 远高于管理决策 F1 (0.17)，揭示了信息获取能力的显著差距。

### 4. SubFit：子模块级 LLM 压缩
**From Layers to Submodules: Rethinking Granularity in Replacement-Based LLM Compression**
📄 [arXiv:2606.02559](https://arxiv.org/abs/2606.02559)

**问题：** 现有 LLM 压缩方法以整层为粒度且要求连续选择，过于粗糙——冗余并不局限于连续区域，Attention 和 FFN 的冗余分布也不同。
**方法：** 提出 SubFit，在子模块级别进行非连续选择，每个子模块配备轻量级拟合残差旁路。仅需校准数据，无需重训练。
**效果：** 在 10 个 LLM、5 个稀疏度级别上，25% 稀疏度下保留 84.6% 密集精度（基线 81.6%），困惑度退化仅 2.42x（基线 4.34x）。

### 5. SimSD：扩散语言模型的简单投机解码
**SimSD: Simple Speculative Decoding in Diffusion Language Models**
📄 [arXiv:2606.02544](https://arxiv.org/abs/2606.02544)

**问题：** 扩散语言模型（dLLM）因依赖 mask token 和双向注意力，无法直接使用标准 token 级投机解码。
**方法：** 设计即插即用的 masking 策略，引入参考 token 并调节注意力掩码，使 dLLM 能在单次前向传播中验证草稿 token。无需训练，可与其他加速技术组合。
**效果：** 在 SDAR 系列 dLLM 上实现最高 7.46x 解码吞吐量提升，同时保持甚至提升生成质量。

### 6. SkillHarm：Agent 技能生命周期攻击基准
**SkillHarm: Lifecycle-Aware Skill-Based Attacks via Automated Construction**
📄 [arXiv:2606.02540](https://arxiv.org/abs/2606.02540)

**问题：** 第三方 Agent 技能是脆弱的攻击面，现有研究仅评估单任务执行中的危害。
**方法：** 构建覆盖技能使用全生命周期的攻击基准，包含固定载荷投毒（FPP）和自变异投毒（SMP）两种场景，定义 12 种风险类型。通过编码 Agent 自动构建 879 个攻击样本。
**发现：** 当前 Agent 在 FPP 下攻击成功率高达 86.3%，SMP 下达 69.3%。许多"防御成功"实际只是 Agent 未能读取被投毒文件，而非真正具备抵抗力。

### 7. SafeSteer：局部化在策略蒸馏实现安全对齐
**SafeSteer: Localized On-Policy Distillation for Efficient Safety Alignment**
📄 [arXiv:2606.02530](https://arxiv.org/abs/2606.02530)

**问题：** LLM 安全对齐往往会损害通用能力（对齐税），现有方法需要海量通用数据或辅助奖励模型。
**方法：** 核心洞察——安全特征在输出分布中是稀疏的，对齐只需局部修改。通过激活引导构建安全教师模型，筛选安全 token，将反向 KL 惩罚限制在这些 token 上。
**效果：** 在多种模型上实现安全性与通用能力的更优平衡。
