## 📰 行业新闻

### 1. Anthropic 发布 Claude Sonnet 5 — 性能逼近 Opus 4.8
Anthropic 发布新一代中端模型 Claude Sonnet 5，作为 Sonnet 4.6 的继任者，支持自主规划、工具调用（浏览器/终端）等 Agent 能力，性能接近旗舰 Opus 4.8。API 首月定价 $2/M 输入 tokens，大幅低于 Opus 的 $5/M。Anthropic 称其为"目前最具 Agent 能力的 Sonnet 模型"。
- 📎 [VentureBeat 报道](https://venturebeat.com/technology/anthropic-launches-claude-sonnet-5-at-a-steep-discount-to-its-top-model-as-the-company-races-toward-a-blockbuster-ipo)
- 📎 [Anthropic 官方公告](https://www.anthropic.com/news/claude-sonnet-5)

### 2. Anthropic 同步推出 Claude Science — AI 科学工作台
Anthropic 发布 Claude Science（Beta），定位为"科学家的 AI 工作台"，将分散的科研工具和数据集整合到一个环境，支持自动生成图表（如 3D 蛋白质结构）。Anthropic 强调这不是新模型，而是面向科研场景的应用层工具。
- 📎 [The Verge 报道](https://www.theverge.com/ai-artificial-intelligence/anthropic-claude-science-beta-launch)
- 📎 [Anthropic 官方公告](https://www.anthropic.com/news/claude-science-ai-workbench)

### 3. Google Gemini Spark 登陆 macOS — AI Agent 可直接操作本地文件
Google 将 Gemini Spark AI Agent 正式推送至 macOS Gemini 应用，Agent 可以直接访问和操作电脑上的文件。新增功能包括连接 Google Tasks 和 Keep、集成 Canva 和 Instacart 等第三方应用，以及实时话题追踪。
- 📎 [Google 官方博客](https://blog.google/innovation-and-ai/products/gemini-app/gemini-spark-updates-june-2026/)
- 📎 [The Verge 报道](https://www.theverge.com/ai-artificial-intelligence/941388/gemini-spark-ai-agent-trip-planning)

### 4. 美团开源 LongCat-2.0 — 1.6T 参数编码模型，全程使用国产芯片训练
美团在 MIT 开源协议下开源 LongCat-2.0，这是一个拥有 1.6T 参数的前沿级 AI 编码模型，在 OpenRouter 排行榜上领先，且全部训练过程均使用国产芯片完成，具有重要的战略意义。
- 📎 [VentureBeat 报道](https://venturebeat.com/technology/meituan-open-sources-longcat-2-0-the-1-6t-near-frontier-agentic-coding-model-thats-been-leading-openrouter-trained-entirely-on-chinese-chips)

### 5. DeepSeek 开源 DSpark — LLM 推理加速框架，最高提速 85%
DeepSeek 发布 DSpark 开源框架，通过优化解码过程加速 LLM 推理，最高可提升 85% 的推理速度。实际加速效果取决于输出质量接受率的配合。
- 📎 [VentureBeat 报道](https://venturebeat.com/orchestration/deepseek-open-sources-dspark-a-new-framework-to-speed-up-llm-inference-by-up-to-85)

## 📚 arXiv 论文精选

### 1. Introspective Coupling: Self-Explanation Training Tracks Behavioral Change
**作者:** Zifan Carl Guo et al. (MIT)
**领域:** LLM 可解释性 / 后训练
研究语言模型在被训练生成自我解释时，是否能产生忠实的内省而非表面模仿。研究发现，使用固定反事实解释数据集进行训练，模型的解释仍能忠实地跟踪自身行为变化（包括谄媚和拒绝等行为），即使行为已经偏移。这为可扩展的内省后训练提供了新路径。
- 📄 [2606.32038](https://arxiv.org/abs/2606.32038v1)

### 2. QVal: 低成本评估长程 LLM Agent 的稠密监督信号
**作者:** Sergio Hernández-Gutiérrez et al.
**领域:** Agent 训练 / 强化学习
针对长程 LLM Agent（单轨迹可达上千动作）中仅靠结果奖励过于稀疏的问题，提出 QVal 测试框架——无需训练即可直接评估稠密监督信号的质量。在 21 种方法的评测中发现，简单的 prompt 基线 consistently 优于文献中的新方法。
- 📄 [2606.32034](https://arxiv.org/abs/2606.32034v1)

### 3. RLMF: 元认知反馈强化学习使 LLM 表达忠实的不确定性
**作者:** Gabrielle Kaili-May Liu et al. (Yale NLP)
**领域:** 元认知 / LLM 对齐
针对 LLM "高置信幻觉"问题，提出基于元认知反馈的强化学习（RLMF），利用模型对自身表现的判断来优化偏好排序。RLMF 在忠实校准任务上超越标准 RL 最多 63%，同时提升了模型对自身能力边界的认知。
- 📄 [2606.32032](https://arxiv.org/abs/2606.32032v1)

### 4. When LLMs Read Tables Carelessly: 数据引用错误的测量与减少
**作者:** Yuqing Yang et al.
**领域:** 表格理解 / ACL 2026 Oral
首次系统性评估 LLM 在表格任务中的数据引用错误（DRE）。发现 1.7B-20B 参数模型均会出现此类错误。引入数据引用批评器（critic）可使回答准确率提升最高 12.0%，4B 参数批评模型 DRE 检测 F1 达 78.2%。
- 📄 [2606.32029](https://arxiv.org/abs/2606.32029v1)

### 5. AdaJEPA: 自适应潜空间世界模型
**作者:** Ying Wang, Oumayma Bounou, Yann LeCun, Mengye Ren
**领域:** 世界模型 / 规划 / Meta 体系
提出 AdaJEPA——一种在 MPC 闭环中执行测试时自适应的潜空间世界模型。通过观测到的状态转换作为自监督信号持续校准世界模型，仅需每步一次梯度更新即可显著提升规划成功率。
- 📄 [2606.32026](https://arxiv.org/abs/2606.32026v1)

### 6. SkillComposer: LLM Agent 的生成式技能组合
**作者:** Xinyu Zhao et al.
**领域:** Agent 技能组合 / 自动化
将 Agent 技能组合形式化为结构化序列预测问题，提出 SkillComposer 框架，在单个自回归解码过程中同时确定激活子集、数量和执行顺序。在 GPT-5.2-Codex 和 Gemini-3-Pro 上，通过率分别提升 23.1 和 18.2 个百分点。
- 📄 [2606.32025](https://arxiv.org/abs/2606.32025v1)
