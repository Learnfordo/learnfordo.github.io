## 📚 arXiv 论文精选

### 1. DeepWeb-Bench：前沿深度研究基准，要求大规模跨源证据与长链推理

作者提出 DeepWeb-Bench，一个远超现有基准难度深度研究评测。9 个前沿模型评测显示：检索不是瓶颈（仅占 12-14% 错误），推理与校准失败占 70% 以上；强模型和弱模型的错误模式截然不同。

🔗 [arXiv:2605.21482](https://arxiv.org/abs/2605.21482)

### 2. AutoRPA：LLM 驱动的 GUI 代码合成，将 ReAct Agent 蒸馏为可复用 RPA 函数

AutoRPA 框架通过翻译-构建管线将 ReAct 风格 Agent 的决策逻辑自动蒸馏为稳健的 RPA 函数，token 使用量降低 82%-96%，大幅提升运行效率和可复用性。已被 ICML 2026 接收。

🔗 [arXiv:2605.21082](https://arxiv.org/abs/2605.21082)

### 3. Agent JIT Compilation：延迟优化的 Web Agent 编译执行

提出 Agent JIT 编译方法，将任务描述直接编译为包含 LLM 调用、工具调用和并行化的可执行代码，替代传统逐步截图-执行循环，通过 JIT-Planner 和 JIT-Scheduler 实现最小延迟调度。

🔗 [arXiv:2605.21470](https://arxiv.org/abs/2605.21470)

### 4. CARV：扩散模型教师的计算感知方差缩减框架

NVIDIA 研究者提出 CARV，通过分层 MC 估计器、时间步重要性采样和分层逆 CDF 构造，在 text-to-3D 蒸馏和归因实验中实现 2-3x 有效计算倍增。

🔗 [arXiv:2605.21489](https://arxiv.org/abs/2605.21489)

### 5. LLM 超参数迁移量化：Embedding 层学习率的关键作用

研究发现 μP 相比标准参数化的核心优势来自 Embedding 层学习率最大化——在标准参数化中它成为瓶颈，导致训练不稳定；提高其学习率可大幅平滑训练并改善超参数迁移。

🔗 [arXiv:2605.21486](https://arxiv.org/abs/2605.21486)

### 6. AI-Native 6G 蓝图：从 "Network for AI" 到 "AI for Network"

提出 6G 时代 AI 原生集成愿景，通过基础模型和多 Agent 系统将网络管理统一为多模态多任务优化问题，实现自诊断、自维护、自恢复的智能通信基础设施。已被 KDD 2026 接收。

🔗 [arXiv:2605.21395](https://arxiv.org/abs/2605.21395)

### 7. QuestBench：通过基准构建教学 AI，揭露深度研究系统隐藏失败

学生将学科知识转化为可验证专家级问题构建 QuestBench（256 题，14 个社科人文领域），13 个评测系统平均通过率仅 16.85%，最佳 GPT-5.5 仅达 57.58%。

🔗 [arXiv:2605.21413](https://arxiv.org/abs/2605.21413)
