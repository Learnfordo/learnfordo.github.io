## 📰 行业新闻

### 1. OpenAI 连续发布 GPT-5.5 系列及多项产品更新

OpenAI 本周密集发布：GPT-5.5 Instant（更智能、更个性化）、GPT-5.5-Cyber（网络安全领域 Trusted Access）、Codex Chrome 扩展（可在用户已登录的网站上自动完成任务）、ChatGPT Trusted Contact 安全功能，以及开始在 ChatGPT 中测试广告投放。这是 OpenAI 从纯产品向商业化迈进的重要信号。

### 2. Anthropic 连续发布三项重要研究

Anthropic 5月7-8日连发三篇：**Teaching Claude why**（通过教导推理原因来降低 agent misalignment）、**Natural Language Autoencoders**（将 Claude 内部表征转化为可读文本，可解释性突破）、以及捐赠开源对齐工具 Petri。对齐研究从理论走向工程化实践。

## 📚 arXiv 论文精选

### 1. LiSCP: 轻量风格一致性检测 LLM 生成文本

**论文:** [arXiv:2605.05950](https://arxiv.org/abs/2605.05950) | cs.CL

提出 LiSCP 方法，通过风格一致性图谱结合离散风格特征与连续语义信号，在对抗场景下检测 LLM 生成的文本。跨域检测性能比现有方法提升 11.79%，在混合人-AI 内容场景中也表现稳健。

### 2. Near-Policy Distillation: 加速 LLM 策略蒸馏 8.1 倍

**论文:** [arXiv:2605.05940](https://arxiv.org/abs/2605.05940) | cs.LG, cs.CL

提出 NPD 框架，将学生生成与训练解耦，用 SFT+序列打包替代昂贵的 RL 框架。openPangu-Embedded-1B 以此方法达到 68.73% SOTA，超越更大的 Qwen3-1.7B。策略蒸馏效率提升显著。

### 3. AlphaCrafter: 多 Agent 全栈量化交易框架

**论文:** [arXiv:2605.05580](https://arxiv.org/abs/2605.05580) | cs.AI

提出三 Agent 协作框架（Miner 发现因子、Screener 筛选策略、Trader 执行交易），在 CSI 300 和 S&P 500 上风险调整收益优于所有基线，且跨试验方差最低。提交至 NeurIPS 2026。

### 4. Mise en Place: Agentic Coding 的上下文工程方法论

**论文:** [arXiv:2605.05400](https://arxiv.org/abs/2605.05400) | cs.SE, cs.AI

借鉴烹饪"mise en place"概念，提出三阶段准备方法论：上下文沉淀、协作规格化、任务分解。在黑客马拉松中，2 小时准备使并行 AI agents 快速实现全栈教育平台。提出"上下文流畅度"作为新开发者技能。VibeX 2026 论文。

### 5. SECDA-DSE: LLM 驱动的 FPGA 加速器设计空间探索

**论文:** [arXiv:2605.05920](https://arxiv.org/abs/2605.05920) | cs.AR, cs.AI

将 LLM 集成到 SECDA 硬件-软件协同设计生态中，用 RAG + chain-of-thought 进行推理引导探索，加上反馈循环实现持续改进。在 Zynq-7000 FPGA 上验证可行性。EuroSys'26 Workshop 论文。

### 6. PLMD: 基于扩散模型的目标导向导航地图补全

**论文:** [arXiv:2605.05960](https://arxiv.org/abs/2605.05960) | cs.RO

提出 Plug-and-Play Label Map Diffusion，用 DDPM 为未观测区域生成障碍与语义标签，实现部分观测环境下的目标定位。ICML 2026 论文。

### 7. QuadraSHAP: 高效精确的 Shapley 值计算

**论文:** [arXiv:2605.05870](https://arxiv.org/abs/2605.05870) | cs.LG

将乘积博弈的 Shapley 值精确转化为一维积分，用 Gauss-Legendre 求积实现可证明的精确计算。数百节点即可在数千特征维度下获得高精度估计，是所有测试配置中最快的数值稳定方法。
