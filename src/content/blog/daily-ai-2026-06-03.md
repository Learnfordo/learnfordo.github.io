## 📚 arXiv 论文精选

### 1. VEPO：视觉锚定 token 选择增强多模态视觉推理 RL
**Unlocking Effective Reinforcement Learning for Visual Reasoning via Vision-Anchored Token Selection**
📄 [arXiv:2606.03937](https://arxiv.org/abs/2606.03937)

**问题：** 在视觉推理任务中，传统基于 token 熵的信用分配机制会失效——因为视觉敏感 token 天然具有低熵，被现有 RLVR 方法忽略。
**方法：** 提出 VEPO 框架，通过乘法耦合将视觉敏感性与 token 熵显式整合，将梯度信用重定向到同时具有视觉基础和信息量的 token 上。
**效果：** 在 7B 规模上超越纯熵基线 2.28 分，3B 规模上超越 3.15 分。消融实验验证了方法的合理性。

### 2. IPT：想象力感知 token 增强多模态空间推理
**Imaginative Perception Tokens Enhance Spatial Reasoning in Multimodal Language Models**
📄 [arXiv:2606.03988](https://arxiv.org/abs/2606.03988)

**问题：** VLM 在空间推理中仍难以处理关键信息不可直接观察的情况——如推断遮挡视角、追踪路径、整合部分观察。
**方法：** 引入"想象力感知 token"（IPT），将 VLM 在不同空间配置下会感知到的中间表征外化，构建约 2 万样本的统一评估基准（视角理解、路径追踪、多视角计数）。
**效果：** 在 MVC 任务上准确率提升 3.4%，在 PT 任务上与强闭源模型竞争力相当，且无需推理时生成图像。

### 3. 量化大型推理模型的忠实置信度表达
**Quantifying Faithful Confidence Expression in Large Reasoning Models**
📄 [arXiv:2606.03969](https://arxiv.org/abs/2606.03969) · 代码：[yale-nlp/faithful_lrm](https://github.com/yale-nlp/faithful_lrm)

**问题：** 大型推理模型（LRM）的长链式思考常被用户解读为"深思熟虑、有能力、自信"的证据，但模型的内在置信度与表达置信度之间的校准（Faithful Calibration）仍是持续失败模式。
**方法：** 提出新框架分析 LRM 的语言确定性相对于三种内部不确定性来源（token 概率、隐藏状态、采样一致性）的对齐程度，并设计前缀条件采样控制条件变异。
**发现：** 推理行为并不自动转化为更好的置信度校准；针对非推理模型的 prompt 干预在推理场景中无效；不同置信度估计器对同一推理链产生分歧评估。

### 4. LLM 推理结构的可视化与效率度量
**Reasoning Structure of Large Language Models**
📄 [arXiv:2606.03883](https://arxiv.org/abs/2606.03883) · ICML 2026 & ICLR 2026 Workshop

**问题：** LRM 常用最终准确率或 token 数量评估，但相同分数可能隐藏根本不同的推理结构。
**方法：** 构建逻辑谜题可扩展基准，将非结构化推理链转化为可验证的推理图（claims + dependencies），定义"推理效率"指标量化逻辑流的集中程度。
**发现：** 结构测量能区分 token 数量和准确率混淆的行为，为诊断失败模式和比较推理如何随难度扩展提供实用工具。

### 5. Hedge-Bench：对冲基金分析师真实任务基准
**Benchmarking Agents on Hard, Realistic Tasks Pertaining to Financial Reasoning**
📄 [arXiv:2606.03918](https://arxiv.org/abs/2606.03918) · 代码：[Trata-Inc/trata-hedge-bench](https://github.com/Trata-Inc/trata-hedge-bench)

**问题：** AI Agent 越来越能处理金融分析的机械任务，但真正有价值的是专家级别的开放式推理——现有基准无法捕捉这类问题。
**方法：** 构建 Hedge-Bench 1.0，包含 102 个基于专业对冲基金分析师真实工作轨迹的任务，支持基于专家步骤的确定性评分。
**发现：** 前沿模型和 Agent 在该基准上得分低于 16%，揭示了当前 AI 在复杂金融推理中的巨大差距。

### 6. QUBRIC：协同设计查询与评分标准，超越可验证奖励的 RL
**Co-Designing Queries and Rubrics for RL Beyond Verifiable Rewards**
📄 [arXiv:2606.03968](https://arxiv.org/abs/2606.03968)

**问题：** 基于评分标准的 RL 是有前景的方向，但现有方法将查询分布视为固定——开放式查询产生模糊标准，过度收窄又引入虚构参考。
**方法：** 提出 QUBRIC 框架，协同设计查询与评分标准：教师派生关键点锚定重写为场景化问题，对比评分生成将教师-策略差距转为查询级标准，可学习性过滤保留有信息量的查询-标准对。
**效果：** 在 ArenaHard 上比 SFT 基线提升 5.5 分，并成功迁移到法律、道德、叙事推理三个未见基准（平均 +6.3 分）。
