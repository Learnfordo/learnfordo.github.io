## 📚 arXiv 论文精选

### 1. GEAR：少复制、多定位——基于证据感知强化学习克服长上下文推理中的重复复制问题

**论文标题：** *Copy Less, Ground More: Overcoming Repetitive Copying in Long-Context Reasoning via Evidence-Aware Reinforcement Learning*

**核心贡献：** 识别了大语言模型在长上下文推理中的一个关键失败模式——"重复复制"（repetitive copying），即模型大量从输入中复制文本而非真正解决问题。作者发现该行为在前沿 LLM 中普遍存在，且随上下文长度增加而加剧。提出了 GEAR（Grounding Evidence-Aware Reward）方法，通过奖励与关键证据的重叠、惩罚与无关上下文的重叠来改进 RL 训练，在多个基准上提升高达 +4.6 分。

> 🔗 [arXiv:2607.19345](https://arxiv.org/abs/2607.19345)

### 2. CodeRescue：编码 Agent 的预算校准恢复路由

**论文标题：** *CodeRescue: Budget-Calibrated Recovery Routing for Coding Agents*

**核心贡献：** 针对编码 Agent 的成本控制问题，提出了"恢复路由"（recovery routing）框架。当廉价模型失败时，不是简单升级到昂贵模型，而是判断是否值得继续用廉价模型恢复。使用 Conformal Risk Control（CRC）层实现预算自适应部署，在 GPT-5.4-nano/GPT-5.4 设定下，以 35% 的恢复成本达到与始终升级相当的解决率。

> 🔗 [arXiv:2607.19338](https://arxiv.org/abs/2607.19338)

### 3. Agents in the Wild：从研究到部署

**论文标题：** *Agents in the Wild: Where Research Meets Deployment*

**核心贡献：** 这是一篇教程型论文，系统梳理了基于 LLM 的 Agent 系统从研究原型到生产部署的转型挑战。覆盖推理与规划、多 Agent 协调、评估等关键领域，通过制药发现和金融系统的案例研究，总结了成功 Agent 系统的常见设计模式，包括验证流水线、回退机制和人工介入监督。

> 🔗 [arXiv:2607.19336](https://arxiv.org/abs/2607.19336)

### 4. Appearance Pointers：扩散 Transformer 的多模态区域控制

**论文标题：** *Appearance Pointers -- Multimodal Region Control of Diffusion Transformers*

**核心贡献：** 提出了"外观指针"（appearance pointers）——一种紧凑的 token 机制，通过将文本或图像输入与用户指定的掩码对齐，引导扩散 Transformer（DiT）在正确的空间位置应用正确的外观线索。这是首个在 DiT 中实现模态无关的局部多模态控制接口，无需从头重新训练基础模型，在多项指标上达到或超越特定模态的 SOTA 方法。

> 🔗 [arXiv:2607.19344](https://arxiv.org/abs/2607.19344)

### 5. ISO：RLVR 原生优化栈

**论文标题：** *ISO: An RLVR-Native Optimization Stack*

**核心贡献：** 研究了强化学习与可验证奖励（RLVR）中常被忽视的优化层。发现了"谱继承"（spectral inheritance）现象——RLVR 可以复用基础模型的权重谱，同时通过输入输出奇异帧的变化获得新行为。提出了等谱优化（Isospectral Optimization, ISO）框架，其离线版本 ISO-Merger 可在无需数据、梯度更新或在线策略蒸馏的情况下合并多个共享基础专家的能力。

> 🔗 [arXiv:2607.19331](https://arxiv.org/abs/2607.19331)
