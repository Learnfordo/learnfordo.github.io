## 📚 arXiv 论文精选

### 1. AgentFloor：小模型在 Agent 工作流中能爬多高？
研究引入 AgentFloor 基准——30 个任务的六层能力阶梯，评测 16 个开源模型（0.27B-32B）及 GPT-5。发现小模型足以胜任大多数短期结构化工具调用任务，最强开源模型整体匹配 GPT-5，差距主要在需要持久约束追踪的长程规划任务上。实用建议：用小模型处理日常操作，仅将前沿模型留给深度规划。
🔗 [arXiv:2605.00334](https://arxiv.org/abs/2605.00334)

### 2. LLM 工具使用的"税"：工具未必总是加分
研究揭示 LLM Agent 的工具调用并不总优于原生 CoT——语义噪声下工具增益常被"工具使用税"（协议开销）抵消。提出 G-STEP 轻量推理门控缓解协议错误，但根本改善仍需强化模型内在推理能力。
🔗 [arXiv:2605.00136](https://arxiv.org/abs/2605.00136)

### 3. LOCA：LLM 越狱成功的局部因果解释
提出 LOCA 方法，为每条越狱攻击给出最小可解释因果解释——平均仅需 6 个中间表示修改即可使模型拒绝，而此前方法做 20 个修改仍常失败。为越狱防御提供细粒度 mechanistic 理解。
🔗 [arXiv:2605.00123](https://arxiv.org/abs/2605.00123)

### 4. TUR-DPO：拓扑与不确定性感知的偏好优化
ICML 2026 论文。改进 DPO 使其奖励推理过程而非仅结果，结合语义忠实度、效用与拓扑质量的校准不确定性信号。7-8B 模型上全面超越 DPO，推理任务上匹配甚至超越 PPO，且保持 RL-free 简洁性。
🔗 [arXiv:2605.00224](https://arxiv.org/abs/2605.00224)

### 5. ARMOR 2025：军用场景 LLM 安全基准
首个基于战争法、交战规则和联合伦理条例三大军事准则的 LLM 安全评测，519 条教义性提示覆盖 12 类 OODA 决策分类，评测 21 个商用 LLM 发现军用安全对齐存在严重缺口。
🔗 [arXiv:2605.00245](https://arxiv.org/abs/2605.00245)

### 6. 集体智能体的因果基础：多 Agent 系统中的涌现集体
提出用因果博弈与因果抽象来判定多 Agent 系统何时可被视为统一集体智能体，解决 Actor-Critic 多 Agent 激励难题，并量化评估不同投票机制的集体性程度。为多 Agent AI 安全提供理论工具。
🔗 [arXiv:2605.00248](https://arxiv.org/abs/2605.00248)

### 7. 哈密顿世界模型：生成式世界建模的物理视角
提出将观测编码到结构化相空间，通过哈密顿动力学（含控制、耗散和残差项）演化状态，以改善世界模型的可解释性、数据效率和长程稳定性，同时讨论摩擦、接触等实际挑战。
🔗 [arXiv:2605.00412](https://arxiv.org/abs/2605.00412)

### 8. AEM：多轮 Agent RL 的自适应熵调制
无需额外监督的信用分配方法，将熵分析从 token 层提升到响应层，实现自然从探索到利用的过渡。1.5B-32B 多模型验证有效，在 SWE-bench-Verified 上为 SOTA 增加 1.4%。
🔗 [arXiv:2605.00425](https://arxiv.org/abs/2605.00425)

### 9. IVLR：交错图文推理轨迹用于长程机器人操作
提出交替文本子目标与视觉关键帧的推理轨迹表示，单次多模态 Transformer 自生成全局语义-几何轨迹后缓存供闭环执行。LIBERO-Long 达 92.4% 成功率，去掉轨迹降至 37.7%。
🔗 [arXiv:2605.00438](https://arxiv.org/abs/2605.00438)

### 10. AgentReputation：去中心化智能体声誉框架
FSE 2026 论文。提出三层去中心化声誉框架（任务执行/声誉服务/防篡改持久化），引入上下文条件声誉卡防止跨领域声誉混淆，并提供决策策略引擎支持资源分配和自适应验证升级。
🔗 [arXiv:2605.00073](https://arxiv.org/abs/2605.00073)
