## 📰 行业新闻

### 1. Anthropic 年化收入突破 300 亿美元，增长 80 倍超预期

Anthropic CEO Dario Amodei 在 Code with Claude 开发者大会上透露，公司原计划 10 倍年增长，实际却达到了 80 倍。年化收入从 2024 年 1 月的 8700 万美元飙升至 2026 年 4 月的 300 亿美元。Claude Code 成为史上增长最快的软件产品之一，6 个月内就达到 10 亿美元年化收入。

→ [VentureBeat 原文](https://venturebeat.com/technology/anthropic-says-it-hit-a-30-billion-revenue-run-rate-after-crazy-80x-growth)

### 2. OpenAI 语音模型获得 GPT-5 级推理能力

OpenAI 发布三款新语音模型：GPT-Realtime-2（具备 GPT-5 级推理）、GPT-Realtime-Translate（70+ 语言实时翻译）和 GPT-Realtime-Whisper（语音转文字）。将对话推理、翻译和转录拆分为独立编排组件，128K token 上下文窗口，改变了语音 Agent 的构建方式。

→ [VentureBeat 原文](https://venturebeat.com/orchestration/openai-brings-gpt-5-class-reasoning-to-real-time-voice-and-it-changes-what-voice-agents-can-actually-orchestrate) | [OpenAI 博客](https://openai.com/index/advancing-voice-intelligence-with-new-models-in-the-api/)

### 3. Thinking Machines 展示"交互模型"新范式——AI 从回合制走向实时对话

前 OpenAI CTO Mira Murati 创立的 Thinking Machines 发布了"交互模型"研究预览，采用全双工架构处理 200ms 音视频块，可同时听、说、看。276B 参数 MoE 模型（12B 活跃参数）搭配后台推理模型，实现近实时 AI 对话交互。

→ [VentureBeat 原文](https://venturebeat.com/technology/thinking-machines-shows-off-preview-of-near-realtime-ai-voice-and-video-conversation-with-new-interaction-models)

### 4. Perceptron Mk1：视频分析 AI 模型性能出色，成本骤降 80-90%

初创公司 Perceptron 发布旗舰视频分析推理模型 Mk1，在空间推理和视频基准上匹敌甚至超越 GPT-5 和 Gemini 3.1 Pro，而 API 成本仅 $0.15/M 输入 token——比 Anthropic、OpenAI、Google 便宜 80-90%。原生视频处理 2 FPS，32K token 上下文窗口。

→ [VentureBeat 原文](https://venturebeat.com/technology/perceptron-mk1-shocks-with-highly-performant-video-analysis-ai-model-80-90-cheaper-than-anthropic-openai-and-google)

### 5. OpenAI 成立 Deployment Company，初始投资 40 亿美元助力企业 AI 落地

OpenAI 成立 Deployment Company，帮助企业部署前沿 AI 系统。TPG 领投，Advent、Bain Capital、Brookfield 等联合投资，McKinsey、Capgemini、Bain & Company 等咨询巨头参与。同时收购 AI 咨询公司 Tomoro，带来 150 名前沿部署工程师。

→ [OpenAI 博客](https://openai.com/index/openai-launches-the-deployment-company/)

## 📚 arXiv 论文精选

### 1. SAPO：面向多模态推理的分段对齐策略优化

**Segment-Aligned Policy Optimization for Multi-Modal Reasoning** — 提出将推理步骤而非 token 或完整序列作为策略更新的基本单元，通过分段级价值估计和重要性采样实现语义对齐。实验表明 SAPO 在准确率和训练稳定性上均优于 token/序列级方法。

→ [arXiv:2605.01327](https://arxiv.org/abs/2605.01327)

### 2. NSI：神经符号技能归纳——让 Agent 自我进化为逻辑架构师

**Programmatic Skill Induction with Neuro-Symbolic Learning for Long-Horizon Agentic Tasks** — ICML 2026 论文。将交互轨迹提炼为模块化逻辑程序，合成显式控制流和动态变量绑定，让 Agent 发现"何时"和"为何"行动。从少样本示例归纳技能并灵活适应未见目标。

→ [arXiv:2605.01293](https://arxiv.org/abs/2605.01293)

### 3. Truth or Tribe：Persona Agent 中的群内偏好如何优先于事实

**How In-group Favoritism Prioritize Facts in Persona Agents** — 发现 Persona Agent 对身份相似同伴的错误答案接受率远高于不同同伴，群内偏好随认知复杂度增加而加剧。提出三种干预策略：身份无关指令、结构化反事实推理、异构视角集成。

→ [arXiv:2605.01329](https://arxiv.org/abs/2605.01329)

### 4. Faithful-Agent：忠实优先的移动 GUI Agent 框架

**Faithful Mobile GUI Agents with Guided Advantage Estimator** — 提出"忠实优先"框架，让 GUI Agent 基于屏幕证据而非记忆捷径行动。两阶段训练：忠实 SFT + 引导优势估计器 (GuAE) RFT，将陷阱成功率从 13.88% 提升至 80.21%。

→ [arXiv:2605.01208](https://arxiv.org/abs/2605.01208)

### 5. Valley3：面向全球电商的全能多模态大模型

**Scaling Omni Foundation Models for E-commerce** — Valley3 支持文本、图像、视频、音频的统一理解与推理，原生多语言音频能力针对电商短视频场景。四阶段训练逐步获得音频理解、跨模态指令跟随、电商知识和长上下文推理能力，配备 Agent 搜索工具用于深度研究。

→ [arXiv:2605.01278](https://arxiv.org/abs/2605.01278)

### 6. DiagramNet：系统级图表识别超越 GPT-5 和 Claude Sonnet 4

**An End-to-End Recognition Framework and Dataset for Non-Standard System-Level Diagrams** — 首个系统级图表多模态数据集，3B 模型搭配多 Agent 工作流在端到端评估中超越 GPT-5、Claude-Sonnet-4 和 Gemini-2.5-Pro 超过 2 倍。工作流使 Gemini-2.5-Pro Task 1 性能提升 128.7 倍。

→ [arXiv:2605.01338](https://arxiv.org/abs/2605.01338)

### 7. EO-Gym：多模态交互式地球观测 Agent 环境

**A Multimodal, Interactive Environment for Earth Observation Agents** — 将地球观测分析建模为 Gymnasium 式交互环境，660k+ 多模态文件、35 个 EO 工具、6 大任务族。微调 Qwen3-VL-4B 使 Pass@3 从 0.49 提升至 0.74，揭示通用 VLM 在交互式 EO 推理上的短板。

→ [arXiv:2605.01250](https://arxiv.org/abs/2605.01250)
