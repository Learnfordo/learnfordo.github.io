## 📰 行业新闻

### 1. OpenAI 模型自主推翻离散几何核心猜想

OpenAI 内部通用推理模型自主推翻了 Erdős 1946 年提出的平面单位距离问题猜想——近 80 年来数学界普遍认为方格构造是最优的，但该模型找到了一组无限族构造实现了多项式级改进。Fields 奖章得主 Tim Gowers 称其为"AI 数学里程碑"，这是 AI 首次自主解决一个子领域的核心开放问题。证明引入了代数数论的精巧技巧。

🔗 [OpenAI Blog](https://openai.com/index/model-disproves-discrete-geometry-conjecture/)

### 2. Cohere 发布 Command A+：首个 Apache 2.0 全开源前沿模型

Cohere 推出 Command A+，218B 参数 Sparse MoE 模型（仅 25B 活跃参数），以 Apache 2.0 许可证完全开源。核心技术突破包括无损 W4A4 量化（单 B200 GPU 或双 H100 即可运行）、原生引用生成（每条事实自动链接源文档），以及 48 语言高效率分词器。在 𝜏²-Bench Telecom 从 37% 跃升至 85%，AIME 25 数学 90%。

🔗 [VentureBeat](https://venturebeat.com/technology/cohere-cracks-lossless-quantization-and-native-citations-with-first-full-apache-2-0-licensed-open-model-command-a)

### 3. Cerebras 运行万亿参数模型速度比 GPU 云快近 7 倍

Cerebras 在 Wafer-Scale Engine 3 上运行 Moonshot AI 的万亿参数 Kimi K2.6 模型，达到 981 tokens/s，经 Artificial Analysis 独立验证——比最快 GPU 云供应商快 6.7 倍，比中位数快 23 倍。标准 agentic coding 请求完整响应仅 5.6 秒，而 Kimi 官方端点需 163.7 秒。

🔗 [VentureBeat](https://venturebeat.com/technology/cerebras-says-its-chips-run-a-trillion-parameter-ai-model-nearly-7-times-faster-than-gpu-clouds)

### 4. Anthropic 发布 Project Glasswing 安全工具扩展

Anthropic 将 Claude Mythos Preview 的安全工具（skills、Claude harness、威胁模型构建器）向更多合格客户开放，并发布开源漏洞披露仪表板。这是 Anthropic 在 Agent 安全透明度上的最新举措。

🔗 [Anthropic Research](https://www.anthropic.com/research/glasswing-initial-update)

### 5. Google 推出 Gemini Managed Agents API

Google I/O 上发布 Managed Agents，一次 API 调用即可部署 AI Agent——集成沙盒、执行循环和基础设施管理。与 Anthropic 的模型层编排、AWS 的授权层方案形成三种不同的 Agent 栈架构路线。

🔗 [VentureBeat](https://venturebeat.com/orchestration/googles-managed-agents-api-promises-one-call-deployment-at-the-cost-of-execution-layer-control)

### 6. Resolve AI 多 Agent 协作诊断生产故障

Resolve AI 推出多 Agent 调查系统，多个专业 Agent 并行追查假设、互相验证结论、构建完整因果链。根因准确性较之前提升 2 倍，DoorDash 报告 MTTR 降低 87%。还引入"背景 Agent"持续监控部署和配置漂移。

🔗 [VentureBeat](https://venturebeat.com/technology/resolve-ai-says-the-ai-coding-boom-is-breaking-production-systems-it-wants-to-fix-that)

## 📚 arXiv 论文精选

### 1. Quantifying and Mitigating Premature Closure in Frontier LLMs

评估五个前沿 LLM 在医学任务中的"过早闭合"倾向——在正确选项被移除时模型仍 55-81% 选择给出答案，对抗性查询中 78% 给出不恰当回答。安全导向提示可减少但无法消除此问题。

📄 [arXiv:2605.15000](https://arxiv.org/abs/2605.15000) · cs.CL, cs.AI

### 2. How to Scale Mixture-of-Experts: From muP to the Maximally Scale-Stable Parameterization

为 MoE 架构三种缩放场景（宽度/专家数/并行缩放）开发 DMFT 描述，推导出 Maximally Scale-Stable Parameterization (MSSP)，解决了 muP 在 MoE 中的学习率迁移和单调改进失效问题。

📄 [arXiv:2605.14200](https://arxiv.org/abs/2605.14200) · cs.LG, stat.ML

### 3. Forecasting Pathways to Scientific Discovery (SciPaths)

提出科学发现路径预测任务：给定目标贡献和前序文献，识别所需的使能贡献并回溯到已有工作。SciPaths 基准含 262 专家标注路径，最佳模型 F1 仅 0.189，核心方法论依赖最难恢复。

📄 [arXiv:2605.14600](https://arxiv.org/abs/2605.14600) · cs.CL

### 4. Dual-Dimensional Consistency: Balancing Budget and Quality in Adaptive Inference-Time Scaling

提出 DDC 框架，联合置信加权贝叶斯协议和趋势感知分层剪枝，桥接推理路径质量与自适应终止。在 5 个基准上减少 10 倍以上 token 消耗，同时保持或超越基线精度。

📄 [arXiv:2605.15100](https://arxiv.org/abs/2605.15100) · cs.AI

### 5. Coreset-Guided Personalized Summarization of Knowledge Graphs (COREKG)

将 coreset 理论应用于个性化 KG 摘要，基于用户查询行为的灵敏度采样，在 Freebase/WikiData/DBpedia 上超越 GLIMPSE、PPR 等方法，仅需原图极小片段。已获 IJCAI 2026 接收。

📄 [arXiv:2605.14900](https://arxiv.org/abs/2605.14900) · cs.AI

### 6. A Lightweight Depth-Enhanced Vision-Language-Action Model

轻量级深度增强 VLA 模型，从 RGB 观察隐式建模 3D 空间感知，无需额外深度传感器或大型几何基础模型，降低训练和部署成本。

📄 [arXiv:2605.14950](https://arxiv.org/abs/2605.14950) · cs.RO

### 7. An Agentic Pipeline for Natural Language-Driven Robot Simulation Environment Synthesis (SR-Platform)

SR-Platform 将自然语言描述转换为可执行的 MuJoCo 仿真环境，四阶段 Agent 流水线（LLM 编排→资产锻造→布局→MJCF 组装），生产部署 611 次调用，五物体场景中位延迟约 50 秒。

📄 [arXiv:2605.14700](https://arxiv.org/abs/2605.14700) · cs.RO
