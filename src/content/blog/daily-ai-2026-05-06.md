## 📚 arXiv 论文精选

### 1. Terminus-4B：小模型能否替代前沿 LLM 执行 Agent 任务？
研究提出 Terminus-4B（基于 Qwen3-4B 微调），专门用于 Agent 终端执行子任务。实验表明该模型在 SWE-Bench Pro 等基准上不仅缩小了与 Claude/GPT-5.3 的差距，某些场景甚至超越前沿模型，同时为主 Agent 减少约 30% 的 token 消耗。
🔗 [arXiv:2605.03195](https://arxiv.org/abs/2605.03195)

### 2. Stop Automating Peer Review：AI 审稿的蜂群效应与可操纵性
ICML 2026 Spotlight 论文指出 AI 审稿系统存在两大问题：1）AI 审稿人呈现"蜂群效应"，过度一致导致视角多样性丧失；2）仅通过 LLM 重写论文风格即可显著提升 AI 评分，证明 AI 审稿极易被操纵。
🔗 [arXiv:2605.03202](https://arxiv.org/abs/2605.03202)

### 3. CreativityBench：评估 LLM 创造性工具使用能力
研究者构建了包含 4K 实体、150K+ 功能标注的大型知识库，生成 14K 任务来评估 LLM 的创造性问题解决能力。评测 10 个前沿模型发现：模型能选出合理物体，但在识别正确部件和物理属性方面严重不足。
🔗 [arXiv:2605.02910](https://arxiv.org/abs/2605.02910)

### 4. Agent Memory 的内部机制：从涌现到诊断的回路分析
对 Qwen-3 系列（0.6B-14B）进行内部回路追踪，发现三个关键现象：0.6B 模型已有路由回路但内容回路在 4B 才出现；Write/Read 共享晚期层作为上下文接地基底；内容回路 4B 可检测但 8B 才可靠可操控。该研究实现了 76.2% 的无监督故障定位准确率。
🔗 [arXiv:2605.03354](https://arxiv.org/abs/2605.03354)

### 5. LLM 旅行规划能力的原子化拆解与评估
将旅行规划拆解为约束提取、工具使用、计划生成、错误识别、错误纠正五个原子能力，发现 LLM 在提取显式约束上表现优秀，但推理隐式开放世界需求困难，且自我纠正存在过度敏感和错误持续的结构性偏差。
🔗 [arXiv:2605.03308](https://arxiv.org/abs/2605.03308)

🔗 [howareu.app](https://howareu.app/)

## 🏢 行业观察

### 8. AI 组织采用中的目标与工人体验鸿沟
通过访谈医疗、金融、管理领域的 AI 日常使用者，研究发现组织对 AI 效率的期望与工人实际体验严重脱节：可用性差、控制权有限、沟通不足是主要障碍。论文提出需在个体、任务、组织三个层面进行适应性调整。
🔗 [arXiv:2605.03078](https://arxiv.org/abs/2605.03078)

### 9. ADAPTS：用多 Agent LLM 架构自动化精神症状评估
提出 ADAPTS 框架，将临床访谈拆解为症状级推理任务，使用混合 Agent LLM 架构进行抑郁症/焦虑症严重度自动评级。在两独立数据集上，自动评级在高分歧访谈中比原始人类评分更接近专家基准（绝对误差 22 vs 26）。
🔗 [arXiv:2605.03212](https://arxiv.org/abs/2605.03212)
