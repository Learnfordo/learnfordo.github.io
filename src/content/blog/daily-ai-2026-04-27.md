## 📄 arXiv 论文速递

### 7. Agentic World Modeling：智能体世界模型的基础、能力与法则

系统综述智能体世界模型（Agentic World Modeling）的理论基础、能力边界与潜在法则，为 Agent 在复杂环境中进行自主决策提供理论框架。

🔗 [arXiv:2604.22748](https://arxiv.org/abs/2604.22748)

### 8. Agent 搜索基准：AgentSearchBench

提出 AgentSearchBench，基于近 10000 个真实 Agent 构建大规模基准，将 Agent 搜索形式化为检索和重排问题。实验揭示了语义相似性与实际 Agent 性能之间的持续差距。

🔗 [arXiv:2604.22436](https://arxiv.org/abs/2604.22436)

### 9. 多 Agent 组织框架：OneManCompany (OMC)

提出 OMC 框架，将多 Agent 系统提升到组织层面。将技能、工具和运行配置封装为可移植的"人才"身份，通过人才市场实现按需招募和动态重组。

🔗 [arXiv:2604.22446](https://arxiv.org/abs/2604.22446)

### 10. Agent 社会集体智能评估：MoltBook 实验

对超过 200 万 Agent 的 MoltBook 平台进行首次大规模集体智能评估，发现 Agent 社会在复杂推理、信息综合和基础协调上均未能超越个体前沿模型，集体智能并未随规模自发涌现。

🔗 [arXiv:2604.22452](https://arxiv.org/abs/2604.22452)
