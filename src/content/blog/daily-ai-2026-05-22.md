## 📰 行业新闻

### 1. OpenAI 模型推翻离散几何核心猜想 — AI 首次自主解决知名数学开放问题

OpenAI 发布了一项重大突破：其内部通用推理模型自主推翻了 Erdős 1946 年提出的平面单位距离问题猜想，构造了无穷族例子实现了多项式级改进。这是 AI 首次自主解决数学子领域核心开放问题。Fields  Medalist Tim Gowers 称其为"AI 数学里程碑"。

🔗 [OpenAI Blog](https://www.openai.com/index/model-disproves-discrete-geometry-conjecture/)

🔗 [TechCrunch](https://techcrunch.com/2026/05/21/trump-delays-ai-security-executive-order-i-dont-want-to-get-in-the-way-of-that-leading/) | [The Verge](https://www.theverge.com/ai-artificial-intelligence)

尽管 SpaceX $15B/年的巨量算力协议仍不够满足 Claude 的需求，Anthropic 正在早期洽谈租用搭载 Microsoft Maia 200 芯片的 Azure 服务器，以扩展 Claude 的运行容量。

### 4. Cohere 发布 Command A+ — 首个 Apache 2.0 开源前沿模型

Cohere 发布了 218B 参数的 Command A+，采用稀疏 MoE 架构（25B 活跃参数），实现近乎无损的 4-bit 量化，可在单张 B200 或两张 H100 上运行。这是首个完整 Apache 2.0 许可的前沿级开源模型，专为推理、多模态文档处理和 Agentic 工作流设计。

🔗 [VentureBeat](https://venturebeat.com/technology/cohere-cracks-lossless-quantization-and-native-citations-with-first-full-apache-2-0-licensed-open-model-command-a)

### 5. Cerebras 跑万亿参数模型速度达 GPU 云近 7 倍

Cerebras 宣布在其晶圆级芯片上运行 Moonshot AI 的万亿参数 Kimi K2.6 模型，输出速度达 981 tokens/s，经 Artificial Analysis 独立验证，比最快的 GPU 云快 6.7 倍。这标志着晶圆级架构首次在万亿参数规模上实现生产级推理。

🔗 [VentureBeat](https://venturebeat.com/technology/cerebras-says-its-chips-run-a-trillion-parameter-ai-model-nearly-7-times-faster-than-gpu-clouds)

### 6. Resolve AI 发布多智能体调查系统，根因定位准确率提升 2x

Resolve AI 推出多智能体协作调查架构，多个专业 Agent 并行追踪多个假设并互相验证结论，构建完整因果链。该系统将根因分析准确率提升 2x，并在 DoorDash 实际部署中将 MTTR 降低 87%。

🔗 [VentureBeat](https://venturebeat.com/technology/resolve-ai-says-the-ai-coding-boom-is-breaking-production-systems-it-wants-to-fix-that)

## 📚 arXiv 论文精选

### 1. Segment-Aligned Policy Optimization for Multi-Modal Reasoning (SAPO)

提出将推理步骤而非 token 或完整序列作为策略更新基本单位的 RL 新范式 SAPO。实验表明 SAPO 在多模态推理任务上显著优于 token 级和序列级策略优化方法，训练稳定性更好。

🔗 [arXiv:2605.01327](https://arxiv.org/abs/2605.01327)

### 2. Programmatic Skill Induction with Neuro-Symbolic Learning for Long-Horizon Agentic Tasks (NSI)

提出 NSI 框架，将交互轨迹提升为逻辑驱动的模块化程序，合成显式控制流和动态变量绑定。ICML 2026 论文，在一系列 Agentic 任务上持续超越 SOTA。

🔗 [arXiv:2605.01293](https://arxiv.org/abs/2605.01293)

### 3. DiagramNet: 非标准系统级图表的端到端识别框架与数据集

首个系统级图表多模态数据集，含 10,977 连接标注和 15,515 CoT QA。结合 3B 模型 + 多 Agent 工作流，超越 GPT-5、Claude-Sonnet-4、Gemini-2.5-Pro 2x+。

🔗 [arXiv:2605.01338](https://arxiv.org/abs/2605.01338)

### 4. Valley3: 面向电商的全模态大模型

面向全球电商场景的全模态 MLLM，统一文本、图像、视频、音频理解。四阶段渐进预训练 + 可控推理模式（1 个非思考模式 + 3 个思考级别）+ Agent 搜索能力。

🔗 [arXiv:2605.01278](https://arxiv.org/abs/2605.01278)

### 5. EO-Gym: 面向地球观测 Agent 的多模态交互环境

将地球观测分析建模为 Gymnasium 风格的交互式工作空间，含 660k+ 多模态文件、35 个 EO 专业工具。微调 Qwen3-VL-4B 后 Pass@3 从 0.49 提升至 0.74。

🔗 [arXiv:2605.01250](https://arxiv.org/abs/2605.01250)

### 6. Llama 用十进制加法推理循环概念

发现 Llama-3.1-8B 在处理循环概念（如"八月后六个月是哪月"）时，并非直接做模加法，而是复用通用十进制加法机制（6+8=14→二月），仅约 0.2% 的 MLP 神经元跨任务共享。

🔗 [arXiv:2605.01148](https://arxiv.org/abs/2605.01148)

### 7. A Cellular Doctrine of Morality: 内在主动精度与心智-现实过载困境

论证当前 AI 系统缺乏内在精度机制来评估信息有效性，导致信息过载和偏见放大。提出基于锥体神经元生物物理动态的内在主动精度方案，可在注意力分配前评估证据充分性。

🔗 [arXiv:2605.01376](https://arxiv.org/abs/2605.01376)
