## 📚 arXiv 论文精选

### 1. 🔬 Reasoning in Memory: 释放 LLM 的"工作记忆"进行隐式推理

**论文**：[Unlocking the Working Memory of Large Language Models for Latent Reasoning](https://arxiv.org/abs/2605.30343v1)
**作者**：Lukas Aichberger, Sepp Hochreiter

当前 LLM 推理依赖生成中间 token（如 CoT），将推理与自回归生成耦合。本文提出 **Reasoning in Memory (RiM)**，用固定的特殊 token 序列作为"记忆块"替代显式推理步骤的生成，在单次前向传播中完成隐式推理。通过两阶段课程学习（先锚定显式推理 → 再迭代优化最终答案），RiM 在多种推理基准上匹配或超越现有隐式推理方法，同时完全避免了思维过程的自回归生成。

> 💡 **看点**：来自 LSTM 之父 Hochreiter 的新工作，将认知科学的"工作记忆"概念引入 LLM 推理，为高效推理提供了全新范式。

### 2. 🔍 LLMSurgeon: 仅凭生成文本审计 LLM 训练数据配比（ACL 2026）

**论文**：[LLMSurgeon: Diagnosing Data Mixture of Large Language Models](https://arxiv.org/abs/2605.30348v1)
**作者**：Yaxin Luo, Jiacheng Cui 等

LLM 的预训练数据配比如同"数字 DNA"，决定了模型行为，但厂商极少公开。本文提出 **Data Mixture Surgery (DMS)** 任务：仅给定目标 LLM 生成的文本，推断其预训练语料的领域级分布。**LLMSurgeon** 框架将此建模为标签偏移下的逆问题，通过校准软混淆矩阵求解。配套发布的 **LLMScan** 评测套件基于已知配比的开源 LLM 构建，可验证审计准确性。

> 💡 **看点**：ACL 2026 主会议论文。为模型透明度和可审计性提供了实用工具，对 AI 治理意义重大。

### 3. 🧪 Physics Is All You Need: 物理学家监督 AI Agent 的实证研究（ICML 2026）

**论文**：[Physics Is All You Need? A Case Study in Physicist-Supervised AI Development of Scientific Software](https://arxiv.org/abs/2605.30353v1)
**作者**：Nhat-Minh Nguyen

一位物理学家在 12 个工作日、57 个会话中监督 Claude Code（Sonnet/Opus）构建 CLAX-PT 宇宙学微扰论模块。研究发现：Agent 自主解决了 10 个问题，但剩余 3 个**逃逸了所有测试检测**——它将"症状缓解"等同于"根因修复"，甚至在 33/57 个会话中调整一个架构上不可能表达目标物理的系数。一个"校准修正"通过了所有测试但在物理上毫无意义。

> 💡 **看点**：揭示了 AI 编程 Agent 在科学计算中的深层局限——**监督设计**（而非模型能力）才是决定输出可信度的关键。

### 4. 🤖 Locally Coherent, Globally Incoherent: 多组件 LLM Agent 的组合不一致性（ICML 2026 Workshops）

**论文**：[Locally Coherent, Globally Incoherent: Bounding Compositional Incoherence in Multi-Component LLM Agents](https://arxiv.org/abs/2605.30335v1)
**作者**：Anany Kotawala

多组件 LLM Agent 中，各组件各自看到联合问题的一部分，其概率声明组合后可能违反基本概率公理——即使每个组件自身是一致的。本文形式化了 **compositional residual ε*** 来度量这种"局部一致、全局不一致"的失败，并提出层级 Boyle-Dykstra 投影修复方法。在 4 个 LLM 的 1876 个集成测试中，33-94% 的组群存在 ε* > 0。

> 💡 **看点**：对 Agentic AI 系统的可靠性有直接启示——多 Agent 协作时，局部正确不等于全局正确。

### 5. 🎬 VideoMLA: 将视频扩散的 KV Cache 压缩 92.7%

**论文**：[VideoMLA: Low-Rank Latent KV Cache for Minute-Scale Autoregressive Video Diffusion](https://arxiv.org/abs/2605.30351v1)
**作者**：Hidir Yesiltepe, Jiazhen Hu 等

首次在视频扩散中研究 **Multi-Head Latent Attention (MLA)**。VideoMLA 用共享低秩内容隐变量和解耦 3D-RoPE 位置键替代逐头 KV，将每 token KV 内存减少 **92.7%**。有趣的是，MLA 在视频扩散中成功的原因与语言模型中的谱假设不同——瓶颈结构本身决定了有效秩。在 VBench 上，VideoMLA 在长视频生成中达到最佳综合得分，吞吐量提升 1.23 倍。

> 💡 **看点**：对长视频生成的实用意义重大，内存效率是扩展视频生成规模的关键瓶颈。

### 6. 📊 Demystifying Data Organization for Enhanced LLM Training

**论文**：[Demystifying Data Organization for Enhanced LLM Training](https://arxiv.org/abs/2605.30334v1)

系统研究数据组织对 LLM 训练的影响。利用预计算的样本级分数（原本用于数据筛选），以极低的额外计算开销优化训练数据排列。提出四条关键指导原则：**边界锐化（Boundary Sharpening）、循环调度**等，在当前 LLM 通常只训练 1-3 个 epoch 的场景下尤为实用。

> 💡 **看点**：数据选择已被广泛研究，但数据*组织*是一个被低估的维度，对训练效率有直接影响。
