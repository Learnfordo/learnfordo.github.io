## 📚 arXiv 论文精选

### 1. Segment-Aligned Policy Optimization (SAPO)：面向多模态推理的强化学习新范式

**arXiv:2605.01327**

现有 RL 方法通常在 token 或完整序列粒度上进行策略优化，与推理过程的自然步骤结构不匹配。SAPO 提出将推理步骤而非 token 作为策略更新的基本单元，引入步骤级 MDP 抽象、段级价值估计和优势计算，在推理基准测试中持续优于 token 级和序列级方法。

🔗 [论文链接](https://arxiv.org/abs/2605.01327)

### 2. Neuro-Symbolic Skill Induction (NSI)：面向长时序 Agent 任务的可编程技能诱导

**arXiv:2605.01293 | ICML 2026**

基于 Foundation Model 的 Agent 在长时序规划中常因纯提示推理的短暂性而失败。NSI 框架将交互轨迹提升为模块化的逻辑程序，通过合成显式控制流和动态变量绑定，使 Agent 发现何时和为何行动，在多个 Agent 任务上持续优于 SOTA 方法。

🔗 [论文链接](https://arxiv.org/abs/2605.01293)

### 3. DiagramNet：面向系统级电路图识别的端到端框架

**arXiv:2605.01338**

系统级电路图编码了芯片设计的架构蓝图，但非标准化符号和训练数据稀缺限制了多模态 LLM 的识别能力。DiagramNet 提出首个系统级电路图多模态数据集（10,977 连接标注 + 15,515 CoT QA），配合渐进训练管线和去耦多 Agent 工作流，3B 模型 + 工作流超越 GPT-5 和 Claude-Sonnet-4。

🔗 [论文链接](https://arxiv.org/abs/2605.01338)

### 4. Valley3：面向电商场景的全能多模态大语言模型

**arXiv:2605.01278**

Valley3 是面向全球电商任务的全能多模态 LLM，统一理解文本、图像、视频和音频。核心特性包括原生多语言音频能力、四阶段持续预训练管线、可控推理模式（非思考 + 三级思考深度），以及 Agent 搜索能力用于电商深度研究。

🔗 [论文链接](https://arxiv.org/abs/2605.01278)

### 5. EO-Gym：面向地球观测 Agent 的多模态交互环境

**arXiv:2605.01250**

地球观测分析本质上是交互式的，但现有基准将其简化为固定输入单轮任务。EO-Gym 提出基于 Gymnasium 的多模态工具使用 Agent 环境，包含 660k+ 多模态文件、35 个 EO 专用工具、9,078 条轨迹。评测显示强通用模型在交互式 EO 推理上仍有显著困难。

🔗 [论文链接](https://arxiv.org/abs/2605.01250)

### 6. Effect-Transparent Governance：AI Workflow 架构的治理形式化

**arXiv:2605.01030**

用 Rocq 8.19 对 AI Workflow 架构的治理进行机器验证形式化，证明效果级治理可在不降低内部计算表达性的前提下施加。36 个模块、~12,000 行 Rocq、454 个定理，0 个待证引理。核心结论：治理与计算表达性是正交维度。

🔗 [论文链接](https://arxiv.org/abs/2605.01030)

### 7. LLMs 尚不应被赋予决策解释的信用

**arXiv:2605.01164**

立场论文：区分决策预测、理由生成和决策解释三个不同证据负担的声明。当前证据支持前两个，但不区分决策解释与预测支持性合理化。提出"信用校准"原则：LLM 应仅被赋予其证据所能支持的最强声明级别的信用。

🔗 [论文链接](https://arxiv.org/abs/2605.01164)
