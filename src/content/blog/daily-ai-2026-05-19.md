## 📚 arXiv 论文精选

### 1. Actionable World Representation (WorldString)
**arXiv:2605.18743** | cs.AI
提出 WorldString 神经架构，能从点云或 RGB-D 视频流直接学习现实物体状态流形的建模，作为物理世界模型的基础构建块，实现可操作的对象表征。
🔗 [论文链接](https://arxiv.org/abs/2605.18743)

### 2. What Does the AI Doctor Value? 审计语言模型临床伦理多元性
**arXiv:2605.18738** | cs.AI
对 LLM 在临床伦理决策中的价值取向进行系统性审计，考察不同模型在面对多元伦理困境时是否体现了足够的多元性与代表性。
🔗 [论文链接](https://arxiv.org/abs/2605.18738)

### 3. SkillGenBench: 基准测试 LLM Agent 技能生成管道
**arXiv:2605.18693** | cs.AI
当 LLM Agent 从"使用技能"转向"生成技能"时，如何评估生成的技能是否正确、可复用且可执行？SkillGenBench 提供了首个系统性的基准测试框架。
🔗 [论文链接](https://arxiv.org/abs/2605.18693)

### 4. 安全部署 LLM Agent 的三层概率 Assume-Guarantee 架构
**arXiv:2605.18672** | cs.AI
论证单层安全护栏对于部署的 LLM Agent 是"结构性不足"而非"临时局限"，提出三层合约架构（语义合规→环境验证→动力学可行性）并通过概率链式法则推导系统级安全边界。
🔗 [论文链接](https://arxiv.org/abs/2605.18672)

### 5. Democratizing Large-Scale Re-Optimization with LLM-Guided Model Patches
**arXiv:2605.18692** | cs.AI
提出 Agent 式重优化框架，让 LLM 作为运筹专家通过自然语言交互动态支持终端用户——将用户提示翻译为优化模型的结构化更新，并从工具箱中选择合适的重优化技术求解。
🔗 [论文链接](https://arxiv.org/abs/2605.18692)

### 6. DeMem: Agent 记忆的速率-失真框架——记住决策而非描述
**arXiv:2605.10870** | cs.AI
将 Agent 记忆压缩建模为决策中心的速率-失真问题，提出 DeMem 在线记忆学习器，仅在数据证明共享状态会导致决策冲突时才细化分区，证明近极小极大后悔保证，在长 horizon 对话基准上一致优于同预算基线。
🔗 [论文链接](https://arxiv.org/abs/2605.10870)

### 7. Mapping the Reliability Boundary of Self-Verification in Medical VQA
**arXiv:2605.10850** | cs.CV
揭示医学 VLM 自验证的"验证海市蜃楼"现象——验证器因容量耦合过度认同生成器，导致高错误率+高认同偏差共存。知识密集型临床任务陷入最深层海市蜃楼，多轮 actor-verifier 循环会将错误答案锁定。
🔗 [论文链接](https://arxiv.org/abs/2605.10850)
